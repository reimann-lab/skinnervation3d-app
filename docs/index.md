# SkInnervation3D

**SkInnervation3D** is a desktop application that provides a graphical interface for defining and running image-analysis workflows — no coding required.

Typical workflows are typically built in two main stages:

- **Pre-processing** — whole-sample processing: conversion to OME-Zarr, flatfield correction, illumination correction, stitching, and more
- **Analysis** — skin innervation analysis: fiber segmentation, counting, surface fitting, plexus analysis, and more

---

## Overview

The application lets you:

1. Set an **analysis directory** — all outputs and logs will be written here
2. Build a **workflow** by selecting and ordering tasks in the left panel
3. Configure **task parameters** for each step
4. Select the **image** to process from the OME-Zarr dataset
5. **Run** the workflow and monitor progress
